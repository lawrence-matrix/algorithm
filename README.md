# AlgoReview

A lightweight web app for reviewing data structures and algorithms.

## What it includes

- Topic review cards for arrays, linked lists, trees, graphs, sorting, dynamic programming, and more.
- Flashcards for quick concept recall.
- A multiple-choice quiz to test algorithm intuition and complexity knowledge.

## How to use

Open `docs/index.html` in a browser to run the app locally.

## GitHub Pages deployment

This repository includes a GitHub Actions workflow at `.github/workflows/pages.yml` that deploys the static site from the `docs/` folder to GitHub Pages on every push to `main`.

The site is available at:

`https://lawrence-matrix.github.io/algorithm/`

> If GitHub Pages is not yet enabled, ensure Pages is configured to use the `gh-pages` branch or the automatic deployment target created by the workflow.
